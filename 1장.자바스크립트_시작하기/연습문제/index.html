1-1 웹에서 프런트엔드와 백엔드의 역할과 프런트엔드 개발자에 대해서 설명하시오.


정답

프론트엔드는 백엔드와의 데이터 통신을 해서 데이터를 받아와서 화면에 보여주는 역할을 한다. 백엔드는 사용자가 필요로하는 데이터를 저장 및 관리를 하며 전달하는 역할을 한다.



1-2 다음은 '안녕!'을 브라우저 화면에 출력하는 프로그램입니다. 밑줄 친 부분을 채워 프로그램을 완성하시오.


문제

        <id="show"></p>
        <script>
            document.getElementById('__________').__________ = '안녕!';
        </script>
정답

        <id="show"></p>
        <script>
            document.getElementById('show').innerHTML = '안녕!';
        </script>

1-3 다음은 '안녕!'을 알림창에 출력하는 프로그램입니다. 밑줄 친 부분을 채워 프로그램을 완성하시오.

문제

        <script>
            __________('안녕!');
        </script>
정답

        <script>
            alert('안녕!');
        </script>

문제

            __________('안녕!');
        </script>

        <script>
            console.log('안녕!');
        </script>
