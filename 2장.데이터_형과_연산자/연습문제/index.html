Q2-12 다음은 거듭제곱, 증가/감소 연산자가 사용된 프로그램입니다. 콘솔에 출력되는 결과를 주석문 빈 칸에 채워보세요.

문제

        let a=3, b=4, c;
        a %= 10;
        b = a**b;
        c = a-b;
        c++;

        console.log(c); // _____
정답

        let a=3, b=4, c;
        a %= 10; // a의 값을 10으로 나눈 나머지로 업데이트합니다. a의 초기값이 3이므로, 이 연산 후에 a의 값은 여전히 3입니다.
        b = a**b; // a를 b번 거듭제곱한 값을 b에 할당합니다. a의 값이 3이고 b의 값이 4이므로, 이 연산 후에 b의 값은 81입니다.
        c = a-b; // a에서 b를 뺀 값을 c에 할당합니다. a의 값이 3이고 b의 값이 81이므로, 이 연산 후에 c의 값은 -78입니다.
        c++; // c의 값을 1 증가시킵니다. c의 초기값이 -78이므로, 이 연산 후에 c의 값은 -77입니다.

        console.log(c); // a의 값은 3, b의 값은 81, c의 값은 -77이 됩니다.
